1. Core dump epidemiology: fixing an 18-year-old bug

来源：OpenAI Engineering
发布日期：2026-06-30
推荐指数：⭐⭐⭐⭐⭐ (5/5) — 必读，年度级基础设施调试范文

核心概要

OpenAI 的 Rockset 服务（2024 年被收购，现用于 ChatGPT 数据基础设施中的搜索和实时分析）在生产环境遭遇了一系列极其诡异的 C++ 崩溃。崩溃表现为两种形态：函数返回时跳到 NULL 地址，或者 %rsp 栈指针莫名偏移 8 字节——这两种都不是正常应用代码的失败模式，任何合理的软件假设都无法解释。

工程师团队最初采用”医生模式”——逐个检查 core dump，形成假设再逐一排除。这个方向完全走进了死胡同，因为他们不自觉地把两个完全不同的 bug 混为一谈，用 A 的反例推翻 B 的假设，又用 B 的反例推翻 A 的假设，陷入了逻辑死循环。

转折点是方法论的根本转变：从”医生”切换到”流行病学家”。团队用 ChatGPT 生成了一个自动化脚本，批量下载并分析过去一年所有的生产环境 core dump，自动提取寄存器状态并分类标记。一旦拥有了高质量的全量数据集，答案几乎立刻浮出水面——实际上是两个独立的 bug：

  • Bug #1：一台 Azure 物理机的 CPU 硬件损坏，导致寄存器计算出错。所有栈指针偏移崩溃都集中在同一区域、同一台机器。
  • Bug #2：GNU libunwind 中一个存在了 18 年的竞态条件——在异常展开（exception unwinding）过程中，_Ux86_64_setcontext 函数更新 %rsp 后、读取目标 %rip 之前的那一条指令的窗口内，如果恰好收到 OpenAI 频繁使用的 SIGUSR2 信号，内核会在 %rsp-128 处构建信号帧，覆盖掉尚未被读取的 ucontext_t 结构体，导致恢复的指令指针变成 NULL。

文章深入分析了 x86_64 ABI 的红区（red zone）、异常展开机制、信号传递的时序细节，并用费米估算验证了竞态窗口的合理性：窗口约 $10^{-10}$ 秒，信号间隔约 $10^{-2}$ 秒，每次异常展开碰撞概率约 $10^{-8}$，而 Rockset 每秒可抛出 $10^4$ 次异常用于背压控制——在集群规模下完全能解释每天十余次崩溃的频率。

关键洞察

  • 方法论决定成败：最关键的一步不是精妙的汇编阅读或对 ABI 的深度理解，而是构建高质量的全量数据集。在数据清洗之前，团队将两个独立现象混为一个故事，任何推理都无法自洽；数据清洗之后，问题结构立刻清晰。
  • 反直觉的竞态窗口：漏洞窗口只有一条指令宽（约 100 皮秒），却因为 OpenAI 极高的异常频率 × 信号频率 × 处理函数栈使用的叠加，在集群规模下被放大到每天可见。这种”微观上不可能、宏观上必然发生”的故障模式，是大规模基础设施独有的调试挑战。
  • 沉默的依赖链：这个 18 年的 bug 一直存在，但从未被触发。直到 Rockset 同时满足三个罕见条件——高频异常抛出、高频 SIGUSR2 信号、信号处理函数因增加 timer_getoverrun 调用而使用更多栈——才越过了”操作可见”的阈值。这揭示了一个深层问题：广泛使用的开源库中的潜在缺陷，可能在特定工作负载组合下突然激活。
  • 解耦调试与缓解：团队不仅在 libunwind 上游提交了修复补丁，还在运维层面做了多层防御——改进致命信号处理函数以记录寄存器状态、改用 libgcc 的展开器（减少锁争用）、修改控制平面使 VM 尽量复用而非回收（便于追踪坏节点）。

评价

优点：

  • 叙事结构极佳——从”看似不可能的崩溃”到”方法论转变”再到”双 bug 揭晓”，像侦探小说一样层层推进，读起来非常过瘾。
  • 技术深度和可读性兼得。作者在涉及 x86_64 ABI、DWARF 元数据、信号传递机制时都做了恰到好处的解释，既不跳步也不堆术语，即使不是系统编程专家也能跟上主线。
  • 核心教训简洁有力且可迁移：在缺乏可靠数据集时不要过度推理。 这对所有做大规模系统运维的团队都适用。

不足：

  • 文章聚焦于 Rockset 场景，对于不使用频繁信号 + 异常背压组合的系统，这个 bug 的复现条件非常苛刻，直接可迁移性有限。但方法论部分（全量数据 vs 个案分析）仍然普适。
  • 硬件故障（Bug #1）部分着墨较少，虽然结论清晰（denylist 一台主机后消失），但缺乏对硬件损坏根因的深入分析。不过这属于篇幅取舍，不算严重缺陷。

适用场景：后端工程师、SRE、系统编程从业者，尤其是处理大规模 C++ 服务稳定性问题的人。对想学习”如何系统性地调试诡异故障”的人来说，这是一篇教科书级别的范文。
